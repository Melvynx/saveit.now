## How To Choose

- For a mobile-first consumer app, start with CapWords, Things, Craft, Paste, or Granola.
- For a creator/productivity app with a demo, start with Screen Studio, Tella, CleanShot X, Screen Charm, or Rotato.
- For a developer/AI app, start with Raycast, Cursor, Linear, Glaze, Spline, or Rive.
- For a playful utility, start with Hot Corners, Mouseless, PixelSnap, or Arc.
- For a viral/original consumer app, start with Cal AI, Finch, Locket, Opal, Daze, BeReal, or Lapse.
- For a health/wellness app, start with Cal AI, Bevel, Bend, Yuka, Sleep Cycle, Clue, Endel, Stoic, or Paired.

Pick one lead reference and at most two support references. Blending more than three usually produces a generic page.
