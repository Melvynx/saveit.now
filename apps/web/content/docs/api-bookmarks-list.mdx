# List Bookmarks

Search and retrieve bookmarks from your SaveIt.now account.

## API Reference

**Method:** `GET`  
**Endpoint:** `/api/v1/bookmarks`

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `query` | string | No | Search query to filter bookmarks |
| `tags` | string | No | Comma-separated list of tags to filter by |
| `types` | string | No | Comma-separated list of bookmark types (PAGE, ARTICLE, VIDEO, etc.) |
| `special` | string | No | Special filters: READ, UNREAD, or STAR |
| `limit` | integer | No | Number of results per page (default: 20, max: 100) |
| `cursor` | string | No | Cursor for pagination (from previous response's nextCursor) |

## Examples

### Bash (cURL)

```bash
curl -X GET "https://saveit.now/api/v1/bookmarks?query=javascript&tags=web,development&types=ARTICLE,PAGE&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### JavaScript

```javascript
const response = await fetch('https://saveit.now/api/v1/bookmarks?query=javascript&tags=web,development&types=ARTICLE,PAGE&limit=10', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json',
  },
```

## Search Functionality

The search query (`query` parameter) supports advanced search across:

- **Title and summary**: Full-text search in bookmark titles and descriptions
- **Content matching**: Semantic search using vector embeddings
- **Tag matching**: Searches through assigned tags
- **Domain matching**: Search by website domain
- **Fuzzy matching**: Returns results even with slight variations in search terms

### Search Examples

```bash
# Search for specific content
curl -X GET "https://saveit.now/api/v1/bookmarks?query=web%20development" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Filter by bookmark type
curl -X GET "https://saveit.now/api/v1/bookmarks?types=ARTICLE,BLOG" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Filter by special attributes
curl -X GET "https://saveit.now/api/v1/bookmarks?special=STAR" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Combined search with tags and types
curl -X GET "https://saveit.now/api/v1/bookmarks?query=javascript&tags=tutorial,guide&types=ARTICLE" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Pagination

Use cursor-based pagination with `limit` and `cursor` parameters:

```bash
# Get first 20 bookmarks
curl -X GET "https://saveit.now/api/v1/bookmarks?limit=20" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Get next 20 bookmarks using cursor from previous response
curl -X GET "https://saveit.now/api/v1/bookmarks?limit=20&cursor=cm4abc123def" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

The response includes:
- `hasMore`: Boolean indicating if more results are available
- `nextCursor`: Cursor value to use for the next page (only present if hasMore is true)

## Notes

- Search is case-insensitive and searches across title, summary, and content
- Results are sorted by relevance score when searching, or by creation date (newest first) when browsing
- Empty search queries return all bookmarks
- Tag filtering supports comma-separated values for multiple tags
- Type filtering supports: PAGE, ARTICLE, BLOG, VIDEO, YOUTUBE, IMAGE, TWEET, PDF
- Special filters: READ (read bookmarks), UNREAD (unread bookmarks), STAR (starred bookmarks)
- Cursor-based pagination ensures consistent results even when new bookmarks are added
